Getting Started

Welcome to Projectr! This is a simple project management tool that allows you to create, view, and manage projects.

To get started, click on the Projects link in the navigation bar to view the list of projects or click the button below. You can click on a project to view its details.

View Projects Create Project

Need Sample Data?

Two sample cities have been pre-loaded into the database: Arglington (High Cost) and Crozet (Low Cost). If you would like to seed the database with sample projects data, you can run the following command from the terminal: 

make loadprojects
